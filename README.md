# NHANES to SDTM Using R

This project converts real-world NHANES clinical data into SDTM-style
datasets using R.

## Domains Created
- DM – Demographics
- VS – Vital Signs
- CM – Concomitant Medications
- AE – Adverse Events

## Data Source
NHANES 2017–2018 (CDC, public data)

## Tools Used
- R
- haven
- dplyr
- tidyr
- sdtm.oak

## Note
Raw NHANES data is not included in this repository.
